## Environment Setup

To run Locally, create a file .env.development and add the following

```
AUTH_DOMAIN=
DATABASE_URL=
STORAGE_BUCKET=
APIKEY=
PROJECTID=
MESSAGE_SENDER_ID=
APP_ID=
MEASUREMENT_ID=
```

## To Run it locally

```
npm install
npm run start
```
